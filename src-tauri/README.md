# CenterMasr Desktop Application

هذه هي النسخة الأصلية من تطبيق ERP لمركز CenterMasr التعليمي، يتم توزيعها كـ **تطبيق Windows واحد** دون الحاجة إلى تثبيت Node.js أو أي تبعيات أخرى.

## ما الذي ستحصل عليه

✅ **تطبيق سطح مكتب كامل بدون علامة مائية** (~30-50 ميجابايت)
✅ **بدون تثبيت** – قم فقط بفتح الملف التنفيذي
✅ **قاعدة بيانات SQLite** – يتم إنشاؤها تلقائيًا في `%APPDATA%/CenterMasr/data.db`
✅ **ترخيص رقمي** – ترخيص لمدة 14 يومًا مع ربط الجهاز بالتوقيع الرقمي
✅ **جميع الميزات** – نفس الواجهة الأمامية React + خادم Express كما هو موجود

## كيفية البدء

### 1. بناء التطبيق
```bash
# التشغيل من المصدر
cargo tauri dev

# أو بناء الإصدار النهائي
cargo tauri build --release
```

### 2. التثبيت (Windows)
1. تشغيل `CenterMasr.exe` (المستقل) أو `CenterMasr-Setup.exe` (مثبت Inno Setup)
2. سيتم فتح المتصفح تلقائيًا على `http://localhost:3001`
3. تسجيل الدخول باستخدام:
   - **اسم المستخدم**: `admin`
   - **كلمة المرور**: `admin123`

### 3. إعادة البناء بعد التحديثات
```bash
# بناء الواجهة الأمامية
cd CenterMasr
npm run build

# إعادة بناء التطبيق الأصلي
cd CenterMasr/src-tauri
cargo tauri build --release
```

## الميزات الرئيسية الأصلية

### 📱 **واجهة المستخدم الكاملة** (مستمدة من React + TypeScript)
- **30+ صفحة** بما في ذلك الطلاب، المدفوعات، الاختبارات، الغياب، التقارير
- **واجهة عربية بالكامل** مع تخطيط من اليمين إلى اليسار
- **تصميم متجاوب** – يعمل على أي شاشة
- **تصميم محمول** – يعمل على الأجهزة اللوحية، أجهزة الكمبيوتر المحمولة

### 🔒 **أمان قوي**
- **ترميز HMAC للترخيص** – يتم ربط الترخيص بتسلسل الأجهزة بشكل مشتق
- **التشفير التام للتطبيق** – لا توجد ملفات خارجية حتى لوحدة التحكم
- **العمل دون اتصال بالإنترنت** – يعمل بالكامل دون اتصال بالإنترنت بعد بدء التشغيل
- **التحقق من الترخيص عند بدء التشغيل** – التحقق من صحة الترخيص قبل تحميل الواجهة الأمامية

### 💾 **التعامل الذكي مع البيانات**
- **sql.js** – قاعدة بيانات SQLite في المتصفح، بدون أي منصة أصلية
- **asm.js fallback** – تحميل أسرع، لا حاجة لـ WASM
- **النسخ الاحتياطي التلقائي** – نسخ احتياطي يومي في `C:\CenterMasrBackup\`
- **الحفاظ على النطاق** – يحافظ على بيانات المشرف أثناء عمليات الاستعادة

### ⚡ **الأداء والأصالة**
- **قاعدة بيانات SQLite المحلية** – قراءة/كتابة فائقة السرعة
- **واجهة برمجة تطبيقات Tauri** – معالجة الأخطاء بشكل أفضل من Electron
- **حجم التطبيق الصغير** – ~30 ميجابايت (مقارنة بـ ~150 ميجابايت مع Electron)
- **قائمة بدء التشغيل** – يعمل عند بدء تشغيل Windows (اختياري)

### 🛠️ **التوزيع easy**
- **ملف تنفيذي واحد** – `CenterMasr.exe` (مستقل) أو `CenterMasr-Setup.exe` (مثبت)
- **لا حاجة للاعتماد الخارجي** – كل شيء مضمن، لا حاجة لـ Node.js، لا حاجة لـ Python، لا حاجة لـ Node-gyp
- **تحديثات التثبيت** – يمكن للمثبت تحديث التطبيق مباشرة

### 📊 **إمكانيات الأم supercomputer**
- **لوحة تحكم SQL الفورية** – استعلامات SQL مع واجهة رسومية
- **تكامل مولد الأرقام التسلسلية** – أكواد الطلاب، الموظفين
- **لوحة التحليلات** – الرسوم البيانية في الوقت الفعلي للمدفوعات، الحضور، الدرجات
- **تتبع الوقت الفعلي** – مراقبة نشاط المستخدم في الخلفية

### 🌐 **الوصول العالمي**
- **بوابات RTL** – تسجيل دخول للآباء، بوابة الطلاب
- **تكامل نظام Windows** – منطقة الإشعارات، قائمة السياق
- **توزيع مثبتة** – يمكن للتوزيع تثبيت الملفات الضرورية، اختصار قائمة ابدأ، تسجيل التشغيل التلقائي
- **حماية كلمة المرور** – كلمة مرور مشرف عالمية على مستوى النظام لاستعادة البيانات

| **حجم قاعدة البيانات**       | PostgreSQL + الملفات | SQLite مضمن |

## البديل: النسخة المستقلة (إذا كنت تفضل بنية Node.js)

إذا كنت ترغب في البنية الأصلية (Node.js + Express) دون تauri، يمكنك الاحتفاظ بملف `CenterMasr.exe` **كحزمة Node.js مستقلة**:

```bash
# استخدام @yao-pkg/pkg (كما هو مذكور في AGENTS.md)
npx @yao-pkg/pkg . --output CenterMasrServer.exe

# أو استخدام Nexe (Node.js runner)
npx nexe server/index.mjs CenterMasrServer.exe
```

كلا الخيارين يعطيانك تجربة مستخدم مماثلة – Tauri فقط تعطي أداء أصلي أفضل وتوزيع أصغر.

## الأسئلة الشائعة

### 🖥️ **هل يعمل هذا على جميع إصدارات Windows؟**
يعمل على **Windows 10+**. بعض الميزات (مثل قائمة ابدأ) متاحة فقط في Windows 10/11.

### 💻 **هل يعمل على أجهزة ARM؟**
نعم، Tauri تدعم ARM. ومع ذلك، فإن sql.js للتصفح فقط يعمل بشكل أسرع على x64.

### 🔄 **ماذا لو قمت بإلغاء تثبيت التطبيق؟**
يقوم Tauri بتخزين البيانات في `%APPDATA%/CenterMasr`. إلغاء التثبيت يترك البيانات فقط – يمكنك إعادة التثبيت لاحقًا.

### 📱 **هل أحتاج إلى اتصال بالإنترنت طوال الوقت؟**
**لا.** يتم تخزين قاعدة البيانات محليًا، ويتم تحميل جميع البيانات عند بدء التشغيل. يعمل التطبيق دون اتصال بالإنترنت تمامًا.

### 🌍 **هل يمكنني تغيير اللغة؟**
نعم، تشمل الواجهة العربية وأيضًا اللغات الدولية (الإنجليزية، الفرنسية، إلخ) عبر إعدادات الواجهة الأمامية.

### 🔑 **ماذا لو فقدت الترخيص؟**
يقوم المثبت بإنشاء **ترخيص تجريبي تلقائي** عند التشغيل لأول مرة (صالح لمدة 14 يومًا). للتجديد الدائم:

2. انتقل إلى **الإعدادات → الترخيص**
